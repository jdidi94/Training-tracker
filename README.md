## Getting Started

### Prerequisites

  ```sh
  npm install npm@latest -g
  ```

  ### Installation
  
 1. Clone the repo
   ```sh
   git clone https://github.com/jdidi94/e-commerce.git
   ```
2. Install NPM packages in the client and the server folders:
   ```sh
   npm install
   ```
4. To run the platform:
   ```sh
   npm start
   ```
